# TechStack used- HTML,CSS,JAVASCRIPT,REACTJS AND REDUX,NODEJS,EXPRESSJS,JSON-WEB-TOKEN,BCRYPT-JS,MONGODB AND MONGOOSE

# To Run Project 
# "npm install" in root folder and inside client folder.
# In root folder "npm run dev"
